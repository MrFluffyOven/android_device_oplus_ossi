# TWRP device tree for Realme GT Neo3 (ossi)
支持 Realme GT Neo 3 150W (ossi) 
## 食用
从 releases 中寻找版本或从代码上编译

### 刷入镜像

- 使用 releases 中压缩包的 bat 文件 (推荐)

```
-- 压缩包内包含了 TWRP 和官方的 Recovery 文件，

-- 下载并解压，双击打开bat文件，按照提示操作即可 (仅能在 bootloader 状态下使用)
```

- 使用 fastboot 命令手动刷入
